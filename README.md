# netflix_clone
🎬 Netflix Clone

A responsive Netflix landing page clone built using HTML, CSS, and JavaScript.
This project recreates the modern Netflix homepage UI with a hero section, trending movies row, smooth hover effects, and a dynamic navbar effect on scroll.

 Features
✅ Responsive Netflix-inspired UI
✅ Fixed navbar with scroll effect
✅ Hero section with background overlay & blur
✅ Trending movies section
✅ Smooth hover animations
✅ Clean and modern design
✅ Pure HTML, CSS, and JavaScript

 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
